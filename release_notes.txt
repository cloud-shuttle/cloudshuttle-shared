Initial release of CloudShuttle shared libraries ecosystem.

## What's Included
- error-handling: Core error types and traits
- observability: Logging, metrics, and tracing utilities
- crypto: Password hashing and encryption
- config: Configuration loading and validation
- validation: Input validation and sanitization
- api: HTTP response formatting and pagination
- auth: JWT authentication middleware (simplified implementation)

## Usage
Add to your Cargo.toml:
```
cloudshuttle-error-handling = { git = "https://github.com/cloud-shuttle/cloudshuttle-shared", tag = "v0.1.0" }
cloudshuttle-observability = { git = "https://github.com/cloud-shuttle/cloudshuttle-shared", tag = "v0.1.0" }
cloudshuttle-crypto = { git = "https://github.com/cloud-shuttle/cloudshuttle-shared", tag = "v0.1.0" }
cloudshuttle-config = { git = "https://github.com/cloud-shuttle/cloudshuttle-shared", tag = "v0.1.0" }
cloudshuttle-validation = { git = "https://github.com/cloud-shuttle/cloudshuttle-shared", tag = "v0.1.0" }
cloudshuttle-api = { git = "https://github.com/cloud-shuttle/cloudshuttle-shared", tag = "v0.1.0" }
cloudshuttle-auth = { git = "https://github.com/cloud-shuttle/cloudshuttle-shared", tag = "v0.1.0" }
```
